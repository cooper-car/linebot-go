# linebot-go

### API格式
| Method       |       URL                         |    說明             | 備註    |
| -------------| --------------------------------- | ------------------ | -------- |
|  POST        | /pushMessage                      |    推送 line 訊息給user   |         |
|  POST        | /callback                         |    儲存 user 訊息到DB     |         |
|  PUT         | /queryMessage                     |    搜尋 user 回傳的訊息    |         |
